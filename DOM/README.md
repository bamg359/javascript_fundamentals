
# Masterclass - Manipulación del DOM con JavaScript

Carpeta inicial para la masterclass. Contiene un `index.html` con la lista de 25 casos prácticos y
archivos de ejemplo en la carpeta `cases/`.

Estructura:
- index.html          -> Página principal con enlaces a los 25 casos
- styles.css          -> Estilos base
- main.js             -> Script principal (vacío de momento)
- cases/              -> Archivos `case01.html` ... `case25.html`
